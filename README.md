Description
-----------
This project is a technical showcase focused on the implementation of a character animation system using Unreal Engine.

It demonstrates the use of animation techniques such as blendspaces, state machines and aim offsets to achieve smooth and responsive character movement.

Controls
-----------
- **W / A / S / D** -> Movement
- **Left Shift** -> Sprint
- **Space** -> Jump
- **Left Ctrl** -> Crouch
- **F** -> Interact / Pick up
- **E** -> Equip weapon
- **G** -> Switch weapon
- **Left Mouse Button** -> Attack / Shoot
- **R** -> Reload
- **V** -> Toggle Melee Mode
- **Mouse** -> Aim  

Animations Implemented
-----------
- Idle / Walk / Run with sprint (Blendspace)
- Crouch system with dedicated animation
- Jump animation integrated with movement states
- Interaction animation (object pickup)
- Weapon equip animation
- Context-based combat system:
  - Ranged attack (available only when weapon is equipped)
  - Melee combat system with 3-hit combo chain
- Reload animation
- Weapon switching system
- Aim Offset system for directional aiming (Up / Down)
  
Features
-----------
- Blendspace-based locomotion system
- Animation State Machine for handling transitions
- Aim Offset system for directional aiming
- Responsive character controls
- Smooth animation blending
